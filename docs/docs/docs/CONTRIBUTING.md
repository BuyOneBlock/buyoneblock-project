# Contributing to Buy One Block

Thank you for your interest in contributing to the Buy One Block ecosystem.

This project follows a strict structure to ensure:
- premium quality,
- legal and regulatory compliance,
- consistency across all documentation and code,
- long‑term stability of the ecosystem.

Please read this document carefully before submitting any contribution.

---

## 1. Code of Conduct

By contributing, you agree to:
- act respectfully and professionally,
- avoid any misleading or speculative communication,
- follow the project’s ethical and compliance standards,
- respect the utility‑token nature of 1BONEBU.

Any contribution violating these principles will be rejected.

---

## 2. Types of Accepted Contributions

We welcome contributions in the following areas:

### ✔ Documentation
- improvements to whitepapers,
- corrections to technical documentation,
- enhancements to clarity, structure, or compliance.

### ✔ Smart Contract Suggestions
- security improvements,
- gas optimization proposals,
- clarity improvements in comments or structure.

### ✔ Tools & Utilities
- scripts that help users interact with the ecosystem,
- non‑financial utilities aligned with the token’s purpose.

### ✔ Branding & UX
- improvements to readability,
- accessibility enhancements,
- non‑speculative communication improvements.

---

## 3. Out‑of‑Scope Contributions

The following contributions **will not be accepted**:

- features that imply financial returns or investment mechanisms,
- governance mechanisms granting token holders decision power,
- speculative tools (price trackers, ROI calculators, etc.),
- integrations with unverified third‑party platforms,
- requests to modify the tokenomics or legal structure.

These restrictions ensure regulatory compliance and protect the project.
